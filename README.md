# Oslo-transit


Dette er et kart som viser hvor T-Bane stopp, Trikke stopp og tog stasjoner i Oslo regionen er.
Du kan også see avganger fra disse stoppene ved å klikke på den. Detter er bare en beta så forvent bugs.
## 🚀 Funksjoner
- Viser alle stopp i Oslo (metro, trikk, tog)
- Du kan klikke på stopp og få neste avganger
- Filter for linjer og transporttyper
- Søkefelt som faktisk funker (ver oppmersom trikk og tebanner som går fra samme sted regnes som to forskjellige stopp)
- Alt oppdateres automatisk hvert 15. sekund
